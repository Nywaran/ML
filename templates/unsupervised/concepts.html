Unsupervised Learning — Basic Concepts

Understanding how machines find patterns in data without labeled examples.

What is Unsupervised Learning?

Unsupervised learning is a type of Machine Learning where the model is trained using data that has no labels or predefined correct answers. Instead of learning from examples, the algorithm finds hidden patterns and structures in the data on its own. It is used when we do not know in advance how the data should be grouped.

What is Clustering?

Clustering is an unsupervised learning technique that groups data points based on their similarity. Points within the same cluster are more similar to each other than to points in other clusters. Clustering is widely used in customer segmentation, anomaly detection, and data compression.

What is K-Means?

K-Means is one of the most popular clustering algorithms. It divides the data into K groups by iteratively assigning each data point to the nearest centroid and then updating the centroids based on the mean of the points in each group. The algorithm repeats this process until the centroids no longer change significantly.

Centroids

A centroid is the center point of a cluster. It is calculated as the mean of all data points assigned to that cluster. In each iteration, centroids are updated to reflect the new center of their group.

Euclidean Distance

The Euclidean distance measures how far apart two points are in space. In K-Means, each data point is assigned to the cluster whose centroid is closest: d = √((x2-x1)² + (y2-y1)²)

Iterative Process

K-Means works in repeated steps: assign each point to the nearest centroid, recalculate centroids, and repeat. The process stops when assignments no longer change between iterations.

Variance and Convergence

Variance (SSE — Sum of Squared Errors) measures how compact the clusters are. As iterations progress, variance decreases. When it stops decreasing, the algorithm has converged and found stable clusters.

Real-World Applications

K-Means is used in customer segmentation to group buyers by behavior, in image compression to reduce the number of colors, in document classification to group similar texts, and in anomaly detection to identify unusual patterns in data.
