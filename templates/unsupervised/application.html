Clustering Application — Customer Segmentation

Grouping customers by age and annual spending using K-Means with a dataset of 1,000 records.

Problem Description

This application uses a dataset of 1,000 customers to identify natural groups based on their age and annual spending. Customer segmentation allows businesses to design targeted marketing strategies for each group.

Dataset Explanation

The dataset contains 1,000 customer records with two numerical features: Age (18 - 70 years) and Annual Spending in USD (500 - 15,000). Data was normalized using StandardScaler before training to ensure both features contribute equally to the distance calculation.

Model Training

A K-Means model was trained with K=3 clusters using Scikit-learn. The model used n_init=10 and random_state=42 for reproducibility.

Interpretation of Results

The model identified three distinct customer segments. The first group consists of younger customers (avg age 26.4) with lower annual spending (avg $2,238). The second group represents middle-aged customers (avg age 43.4) with moderate spending (avg $6,600). The third group contains older customers (avg age 61.3) with the highest spending levels (avg $12,010). These segments can be used to design targeted promotions and personalized marketing strategies for each customer profile.
