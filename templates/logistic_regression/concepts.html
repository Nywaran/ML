Logistic Regression — Basic Concepts

Understanding how logistic regression solves classification problems.

What is Logistic Regression?

Logistic Regression is a supervised learning algorithm used to solve classification problems. Unlike linear regression which predicts a continuous value, logistic regression predicts the probability that an input belongs to a specific class. The output is always a value between 0 and 1, which is then converted into a class label based on a defined threshold.

Difference between Regression and Classification

Regression predicts a continuous numerical value, as demonstrated in the gold price example. Classification predicts a category or class. Logistic regression is a classification algorithm despite having the word regression in its name.

Binary Classification

Binary classification is a type of classification problem where the output can only be one of two possible classes. For example: spam or not spam, fraud or not fraud, default or no default. Logistic regression is specifically designed for this type of problem.

Probability Estimation and Decision Boundary

Logistic regression uses the Sigmoid function to convert any numerical value into a probability between 0 and 1. The decision boundary is the threshold that separates the two classes. By default this is 0.5 — if the predicted probability is greater than or equal to 0.5 the model predicts class 1, otherwise it predicts class 0.

Real-World Use Example

A bank uses logistic regression to predict whether a loan applicant will default on their payment. Based on variables such as income, credit score, and loan amount, the model returns a probability of default and classifies the applicant as high risk or low risk.
