Linear Regression — Basic Concepts

Understanding the fundamentals of one of the most important supervised learning algorithms.

What is Linear Regression?

Linear regression is a supervised learning algorithm that predicts a continuous numerical value based on one or more input variables. It works by finding the best straight line that fits the relationship between the input data and the expected output.

Independent and Dependent Variables

The independent variable is the input — the data we already know. The dependent variable is the output — the value we want to predict. For example, if we want to predict the price of gold based on its trading volume, the volume is the independent variable and the price is the dependent variable.

Regression Line, Slope and Intercept

The regression line is the straight line that best fits the data. It is defined by two values: the slope, which indicates how much the output changes when the input increases by one unit, and the intercept, which is the value of the output when the input is zero.

Training Data

Training data is the dataset used to teach the model. The algorithm analyzes this data to find the line that minimizes the error between the predicted values and the real values.

Example of Application

A simple example: predicting a student's final grade based on the number of hours studied. The more hours studied (independent variable), the higher the expected grade (dependent variable). The regression model finds the line that best represents this relationship.
